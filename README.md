# .github

Organization-wide defaults for [@TodeaEngineering](https://github.com/TodeaEngineering). This repository is **public** — contents are visible to anyone on the internet.

## Contents

- [`profile/README.md`](profile/README.md) — rendered on the organization's
  GitHub landing page. The public-facing description of what Todea is and
  how to engage. Update when positioning, services, or coverage geography
  change.
- [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) — applies to every public
  repository in the organization.
- [`CONTRIBUTING.md`](CONTRIBUTING.md) — default contribution guidance.
  Individual repositories may ship their own `CONTRIBUTING.md` to add
  project-specific notes.
- [`LICENSE`](LICENSE) — Apache 2.0, the default license Todea uses for
  new public projects. Individual repos should still ship their own
  `LICENSE` file; GitHub does not fall back to this one automatically.

## Brand

- Primary surface color: `#0a0f10` (near-black, used on the website footer).
- Logo mark: [`Logo.svg` on todea.co.kr](https://todea.co.kr/Logo.svg) —
  pure black on light backgrounds, inverted on dark surfaces.

## Updating

Changes to `profile/README.md` appear on
[github.com/TodeaEngineering](https://github.com/TodeaEngineering) within
a few minutes of merging to `main`.

## Scope

This repository is intentionally minimal: org-page copy, default community
files, and the org-wide license. Anything beyond that — service catalog,
case studies, internal documentation, contracts — does not belong here.
