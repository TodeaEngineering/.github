# .github

Organization-wide defaults for [@TodeaEngineering](https://github.com/TodeaEngineering).

## Contents

- [`profile/README.md`](profile/README.md) — rendered on the organization's
  GitHub landing page.
- [`LICENSE`](LICENSE) — Apache 2.0, the default license Todea uses for new
  public projects. Individual repos should still ship their own `LICENSE`
  file; GitHub does not fall back to this one automatically.

## Brand

- Primary surface color: `#0a0f10` (near-black, used on the website footer).
- Logo mark: `Logo.svg` on [todea.co.kr](https://todea.co.kr/Logo.svg) — pure
  black on light backgrounds, inverted on dark surfaces.

## Updating

Changes to `profile/README.md` appear on
[github.com/TodeaEngineering](https://github.com/TodeaEngineering) within a
few minutes of merging to `main`.
